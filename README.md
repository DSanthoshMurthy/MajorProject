## StreamlitApp

### Setup

```bash
cd /Users/dsanthoshmurthy/Documents/StreamlitApp
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -U pip
python -m pip install -r requirements.txt
```

### Run

```bash
cd /Users/dsanthoshmurthy/Documents/StreamlitApp
source .venv/bin/activate
streamlit run app.py
```
